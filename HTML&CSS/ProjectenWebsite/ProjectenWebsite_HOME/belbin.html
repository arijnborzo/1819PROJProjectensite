Belbintest

Welkom student

 Omdat je voor het eerst inlogt, hadden we graag geweten wat voor type groeppersoon je bent. Op deze manier hebben je toekomstige teamgenoten al een idee van wie je bent, en heb jij al een beeld van hoe zij zich zullen gedragen in het vak Project + Wetenschappelijk rapporteren. Vul daarom snel deze Belbin test in. Eens je deze hebt ingevuld kan je de projectvoorstellen indienen. Veel succes!

Je ziet hieronder een aantal stellingen die iets zeggen over jouw rol in een team. Geef per stellingen aan in hoeverre deze voor jou kenmerkend is.
